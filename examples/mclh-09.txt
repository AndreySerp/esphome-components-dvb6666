Калибровка датчиков mclh-09

при включении питания долгим нажатием на кнопку производится калибровка датчика уровня освещения и влажности почвы.
Датчик должен быть на свету, электроды  в воздухе, чистые и сухие.
Датчик медленно моргнет три раза и выполнится калибровка по текущему состоянию.
будет зафиксировано 0% влажности почвы и 798 lx освещенности
Если это выполнить при недостатке освещения значения яркости будут сильно завышены. Если электроды датчика будут в грунте - значения влажности будут занижены.

Режим калибровки также можно включить через селектор цвета светодиодов. Режим Green (Once) выполнит калибровку по текущему состоянию влажности и освещенности.

Калибровка температуры проводится при 24,6 гадусах цельсия. Через режим выбор режима Red (Once) в селекторе.

Сброс всех значений на заводские выполняется через выбор в селекторе режима Red + Green (Once)

Значения в IMMEDIATE ALERT -> alert level
01 ( Быстрый красный ) - это калибровка температуры в + 24,6 градуса (688 raw)
02 ( Медленный зеленый ) это калибровка в 0% влажности почвы (900 raw) и яркости в 798 lx в попугаях. (154raw)
03 ( красно-зеленое быстрое мигание ) - сброс на прописанные на заводе значения. 

Калибровка по температуре не сбрасывается после отключения питания или включения/выключения.

Есть еще код ( 00 ) двоякого действия. после включения питания или сброса - двухкратное красное моргание, после мигания зеленым ( 02 )  и отправке кода ( 00 ) постоянное свечение зеленым, в оригинале используется для индикации выхода за пределы по температре, влажности  и непонятно как по освещенности в течении времени ( на усмотрение хоста / шлюза )

коды выше: - ( 04 ) короткое мигание зеленым, (в ноль значения по влажности постоянно)
- ( 05)  короткое мигание зеленым,  (в максимум значения по влажности постоянно)
слегка деструктивны, т.е. не сбрасываются выключением питания, только сбросом. кодом ( 03 ) или красно-зеленым миганием.
